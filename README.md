# Infrastructure Health Agent (Rust)

## Overview
The Infrastructure Health Agent is a lightweight, high-performance system daemon written in Rust designed to run alongside mission-critical database instances. Its primary purpose is to provide critial,performance-sensitive and wide-scale DB systems (like the Azure MySQL Control Plane) with real-time telemetry, process-level health monitoring, and automated system-state reporting for planetary-scale database clusters.
